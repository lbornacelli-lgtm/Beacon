# FPREN — Common Broadcast Standards

## Station
FPREN — Florida Public Radio Emergency Network
Coverage: Alachua, Marion, Levy, Columbia, Gilchrist counties

## Language
- Plain American English, 8th-grade reading level.
- Active voice. 12-hour time format with AM/PM.
- Use "tonight", "tomorrow" over date strings when applicable.

## Never include
- NWS/SAME codes, UGC identifiers.
- URLs or phone numbers unless part of an action message.
- "According to..." attribution phrases.

## Format
Plain prose only. No bullets, headers, or markdown in broadcast output.
